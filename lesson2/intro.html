Lesson 2 - Introduction

Welcome to Lesson 2 - Working With Data. This lesson builds off of the knowledge learned in Lesson 1. In this lesson, we will learn how to handle data within our program.

We will be using JavaScript as our programming language to learn some of the programming fundamentals. Javascript is the programming langauge of the internet, which makes it commonly used, easy to understand, and quick to deploy so you can see instant results as you learn!

Throughout the lessons, we will be using an embedded development tool provided by JS Bin. This will allow us to test out our skills as we learn them!
