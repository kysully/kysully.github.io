Lesson 2 - Part 1: Data

Data is at the essence of what computer scientists handle. Every program is written to perform some operation on some data. This website that you are reading is comprised of various different forms of data that your web browser (the program handling the data) interprets and displays on your screen.

In Lesson 1, we learned about using variables. Now we will learn a new variable which can handle the data we will be using.

We will first look at arrays. An array is a collection of data stored inside one variable. We can declare an array using [] brackets 
var data = [1, 2, 3, 4, 5]; //declares an array named data
console.log(data); //prints out the contents of our data
