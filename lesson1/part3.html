Lesson 1 - Part 3: Working with Variables

Let's continue our introduction by exploring some more details about variables.

Operator Description
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus
++ Increment
-- Decrement 
var a = 1; //declares a variable named a and assigns it the value 1
var b = 2; //sets the value of b to 2
var c = 3.33333 //sets the value of c to the floating point number 1.33333
var word = "cool"

//print out our variables to the console
console.log(a);
console.log(b);
console.log(c);
console.log(word);
