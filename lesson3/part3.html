Lesson 3 - Part 3: Using Functions

Functions are very versitile. A programmer can manipulate the values of their data through functions. Functions often are written for a specific purpose as well. By writing each function to serve a single purpose, a programmer can help to keep their code simplistic.

We can use functions to calculate various math formulas given different data, manipulate data, or many other applications. We can apply functions to revisit our class example from the last lesson.

Let's look at some examples. 
function assignGrade(student, grade){
          student.grade = grade;
        }

function averageAge(roster){
  var average = 0;
  var count = 0;
  roster.forEach(function(student){
    average += student.age;
    count++;
  });
  return average / count;
}

var classRoster = []; //declare an array to store the class roster
var student1 = {name: "Kyle", age: 21, studentID: 1234};
var student2 = {name: "John", age: 22, studentID: 4352};
var student3 = {name: "Mary", age: 20, studentID: 2322};

assignGrade(student1, "A");
assignGrade(student2, "B");
assignGrade(student3, "C");

classRoster[0] = student1;
classRoster[1] = student2;
classRoster[3] = student3;

console.log(classRoster);
console.log(classRoster[1].name + " has a grade of " + classRoster[1].grade);
console.log("The average age is " + averageAge(classRoster));
