Lesson 3 - Part 1: Intro to Functions

Just like data is a collection of various values, a function is a collection of code. In JavaScript, a programmer can declare a function with instructions inside, and later call the function. When the function is called (invoked) later on, the instructions are executed.

When calling a function, you can pass in variables.This means that a programmer can write a function which takes variables, performs some operation on the data within the variables, and return back to the original flow of the program.

Let's check out functions. 
function product(p1, p2) {
    return p1 * p2;              // The function returns the product of p1 and p2
}
var x = product(2, 3);
console.log(x);
